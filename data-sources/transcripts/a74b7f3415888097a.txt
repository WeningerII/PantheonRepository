All 56 faculties termed. Summary of method and key findings:

- **Source file**: `/tmp/powers-roster.json` — keys `Slavic` (6 figures, 28 faculties) and `Finnish` (6 figures, 28 faculties).
- **No skips, no em-dashes used.** Every faculty including titles/offices received a native concept-word. Rulership/office terms supplied where the faculty named a station (emäntä "mistress," bogatyr, boyarin, skotiy bog, etc.).
- **Script tagging**: Slavic divine/Chronicle material uses OCS (Old Church Slavonic) orthography with reconstructions marked `*` (e.g. *zmьjь, *Svarogъ); the East-Slavic bylina heroes use Russian script tags since the byliny are Russian-language. Finnish uses Kalevala/SKVR vocabulary.
- **Web-verified anchors**: Pohjolan emäntä (Louhi's canonical title), skotiy bog (Veles, Primary Chronicle), perun = thunderbolt, Mati Syra Zemlya (Mokosh), Tuonelan joutsen, väki (animating force), bogatyr (< Turkic baɣatur) / druzhina, kantele/tietäjä/loitsu, Svarog forging interpolation (Hypatian s.a. 1114), Sampo.
- **One factual correction flagged inline**: the roster's note names Lemminkäinen's reviver "Lempi," but in the Kalevala his mother is unnamed (the *emo*/mother); I termed the faculty on the native concept (herättäminen) and noted the discrepancy rather than inventing a name.
- **Inheritability**: set `none` for all spheres/titles/offices; `partial`/`full` only for heritable bodily powers (e.g. Kullervo's innate väki = full; Louhi's shape-shifting and weather-conjuring = partial; Aino's fish-transformation = partial).
- Added two optional `+` faculties (Ilya's bow-archery; nothing forced).
