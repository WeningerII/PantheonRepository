All 118 assigned faculties now carry native terms. A few notes on method and judgment calls:

**Coverage:** Every existing faculty id across Yoruba (29), Edo (32), Akan (42), Shilluk (14), Banyaruanda (1) received a native term. None skipped.

**The hard rule on titles/rulership:** Every kingship/office faculty got the native title, never an em-dash — Yoruba ọba / Aláàfin, Edo Oba / Iyoba, Akan ɔhene / Asantehene / Ɔhemaa / Otumfuo / Kumasihene, Shilluk reth, Kinyarwanda umwami.

**Genuine em-dash cases (near-zero, as instructed):** Used only where there is truly no indigenous single word for an introduced/abstract concept: Esigie's court-Christianity (Portuguese import) and Emotan's abstract "conscience" deity-term, and Ewedo's borrowed administrative gloss — in each I still supplied the closest authentic Edo noun (izẹ "righteousness", Eki "market", Eghaevbo "councilors") rather than leaving it bare.

**Inheritability:** Set to none for all spheres/titles/offices. The only heritable bodily/personal powers flagged: Oranmiyan's iron-power from Ogun (partial, lineage-degrading) and Anansi's day-name Kwaku (full — day-names pass by birth-day).

**Orthography:** Yoruba with full tone marks and subdot (Ṣàngó, àṣẹ, ọba, ọdẹ); Akan/Twi with the open-vowel characters ɔ/ɛ and the correct Asante spellings (Ɔkɔmfoɔ, Asantehene, Anansesɛm, Sika Dwa Kofi); Edo with the digraphs and the Met's attested guild spelling Igun Ẹ́rọ̀nmwọ̀n; Shilluk with the ø/ä vowels (reth, jwok, nyaŋ, Pachodo) per Westermann.

**Verification:** Core terms web-confirmed — Ogun/irin/alágbẹ̀dẹ, àṣẹ, Ṣàngó (àrá, Èdùn Àrá, idájọ́, Aláàfin), Edo Oba/Osa/Osanobua/Iya/Iyoba/Igun Ẹ́rọ̀nmwọ̀n/Eko/Iwebo-Iweguae-Ibiwe, Akan Onyame/Onyankopɔn/abosom/sunsum/ɔkra/Sika Dwa/Asantemanhyiamu/Otumfuo/Katakyie/Feyiase, Shilluk reth/Nyikang/jwok/Pachodo (Westermann, Evans-Pritchard, Seligman, Lienhardt), and Kinyarwanda imandwa/kubandwa/Ryangombe/umwami.

**Optional additions appended (+):** three, each marked with the "+<kebab-id>" form — a Shilluk water-spirit (jwok), a Kinyarwanda kubandwa initiation-rite, both with native terms.
